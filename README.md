# vfox-postgres-exporter

[![CI](https://github.com/jbox-web/vfox-postgres-exporter/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-postgres-exporter/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [postgres_exporter](https://github.com/prometheus-community/postgres_exporter)

## Install

```
mise plugin install postgres-exporter https://github.com/jbox-web/vfox-postgres-exporter.git
```
